# 火箭運算程式 - 完整能力說明

本程式是一個**概念設計與教育研究平台**，整合了完整的工程公式庫、Von Kármán 與錢學森理論框架，以及完整的設計流程。

**適用範圍**：
- 概念設計階段性能估算
- 教育與研究用途
- 算法開發與理論驗證

**限制**：
- 未經完整 V&V 驗證，不適用於最終設計
- 部分模型使用簡化假設，需專業工具交叉驗證
- 氣動係數為占位或查表，需真實 CFD/風洞數據

## 🚀 核心能力總覽

### 一、6自由度（6DoF）飛行模擬

#### 1.1 完整動力學模型
- **位置與速度**：3D 空間位置 `r_I`、速度 `v_I`（支援 ECI/ECEF 座標系）
- **姿態**：四元數 `q_IB`（避免萬向節鎖）
- **角速度**：`w_B`（機體座標系）
- **質量變化**：`m`（燃料消耗）
- **熱狀態**：表面溫度 `T_w`、內部溫度 `T_int`
- **節流控制**：`throttle_act`（含動態響應）

#### 1.2 座標系支援
- **ECI（地心慣性）**：標準慣性座標
- **ECEF（地心地固）**：支援地球自轉、Coriolis 效應
- **NED（北東地）**：本地座標系，用於風場計算

#### 1.3 數值積分
- **RK4（四階龍格-庫塔）**：高精度數值積分
- **自動狀態正規化**：四元數自動歸一化
- **邊界保護**：質量、節流等邊界條件處理

### 二、大氣環境模型

#### 2.1 完整 ISA 大氣模型
- **8 層大氣**：從海平面到 86 km
  - 對流層（0-11 km）
  - 平流層（11-47 km）
  - 中間層（47-86 km）
- **完整物理參數**：
  - 溫度 `T`、壓力 `p`、密度 `ρ`
  - 音速 `a`、比熱比 `γ`
  - **黏度 `μ`**（Sutherland 定律）
  - **導熱係數 `k`**（幂律模型）
  - **Prandtl 數 `Pr`**
  - **Reynolds 數 `Re`**、**Nusselt 數 `Nu`**

#### 2.2 風場模型
- **NED 風場**：基於高度的風剪切模型
- **座標轉換**：風場自動轉換到 ECI/ECEF
- **空速計算**：考慮風場影響的真實空速

### 三、氣動力學

#### 3.1 基礎氣動模型
- **升力**：`L = q * S * C_L`
- **阻力**：`D = q * S * C_D`
- **力矩**：`M = q * S * c * C_M`
- **動壓**：`q = 0.5 * ρ * V²`

#### 3.2 可壓縮流
- **等熵關係**：總溫、總壓計算
- **正激波**：壓力比、密度比、馬赫數變化
- **Prandtl-Meyer 展開**：超音速轉角
- **Prandtl-Glauert 修正**：亞音速壓縮性
- **Kármán-Tsien 修正**：更精確的可壓縮性修正

#### 3.3 氣動係數模型
- **占位模型**：線性升力、拋物線阻力
- **查表模型**：2D 插值（alpha × M）
- **代理模型介面**：可替換為 CFD/風洞數據
- **Re 修正**：可選的雷諾數修正因子

#### 3.4 進階氣動
- **阻力分解**：皮膚摩擦、形狀、波阻、誘導阻力
- **升力線理論**：有限翼升力計算
- **靜穩定裕度**：`SM = (x_NP - x_CG) / c`
- **壓力係數**：`C_p = (p - p_∞) / q_∞`

### 四、推進系統

#### 4.1 化學推進
- **推力方程**：`F = ṁ*v_e + (p_e - p_a)*A_e`
- **等熵噴管**：
  - 面積比-馬赫數關係
  - 出口壓力、溫度、速度計算
  - 最佳膨脹比
- **燃燒室模型**：
  - 特徵速度 `c*`
  - 推力係數 `C_F`
  - 燃燒效率
- **節流動態**：一階響應 `d(throttle)/dt = (cmd - act)/τ`
- **低壓修正**：真空與大氣環境推力差異

#### 4.2 電推進
- **離子推進**：`v_e = √(2*q*V_acc / m_i)`
- **電熱推進**：`v_e = √(2*c_p*(T_e - T_a))`
- **電磁推進**：磁場推力近似
- **效率分解**：`η_T = η_e * η_m * η_c`
- **功率-推力關係**：`F = 2*η*P_in / v_e`

#### 4.3 脈衝推進
- **脈衝等離子（PPT）**：衝量位元、平均推力
- **脈衝感應（PIT）**：脈衝能量與推力
- **脈衝頻率**：`F_avg = I_b * f`

#### 4.4 核熱推進
- **核熱排氣速度**：`v_e ≈ √(2*γ*R*T_h)`
- **核電推進**：核能發電驅動電推進
- **核脈衝**：爆炸波推進模型

#### 4.5 推進性能指標
- **比衝**：`I_sp = F / (ṁ * g0)`
- **等效排氣速度**：`V_eq = v_e + (p_e - p_a)*A_e / ṁ`
- **推重比**：`T/W = F / (m * g0)`
- **推進效率**：`η = I_sp,actual / I_sp,ideal`
- **噴管效率**：`η_nozzle = v_e,actual / v_e,ideal`

### 五、熱防護系統（TPS）

#### 5.1 對流加熱
- **Sutton-Graves 模型**：`q̇ = k * √(ρ/R_n) * V³`
- **Nu/Re 修正**：可選的 Nusselt 數模型
- **層流平板**：`Nu = 0.332 * Re^0.5 * Pr^(1/3)`

#### 5.2 輻射散熱
- **Stefan-Boltzmann**：`q_rad = ε*σ*(T_w⁴ - T_env⁴)`

#### 5.3 能量平衡
- **表面節點**：對流、輻射、導熱平衡
- **內部節點**：導熱與內部對流
- **1D 導熱**：簡化熱傳模型

#### 5.4 高超音速熱流
- **Kármán-Sutton 型**：再入熱流估算
- **高焓氣流**：總焓計算
- **化學反應效應**：非平衡化學模型

### 八、軌跡分析與最佳化

#### 8.1 錢學森彈道（三方程）
- **速度方程**：`dV/dt = (T*cos(α) - D)/m - g*sin(γ)`
- **航跡角方程**：`dγ/dt = (T*sin(α) + L)/(m*V) - (g*cos(γ))/V`
- **高度方程**：`dh/dt = V*sin(γ)`
- **質量方程**：`dm/dt = -ṁ_fuel`

#### 8.2 最優彈道
- **最長射程彈道**：分段控制策略
  - 初期：小攻角快速增速
  - 中段：零升力滑翔
  - 末段：調整攻角
- **最小燃料彈道**：最佳控制問題
  - 目標：`min ∫ ṁ dt`
  - 考慮氣動、重力、推力限制

#### 8.3 軌跡最佳化
- **成本函數**：最小燃料、最小時間、最大射程
- **Pontryagin 最優性**：Hamiltonian 計算
- **啟發式方法**：快速近似解

#### 8.4 Δv 分析
- **理想 Δv**：火箭方程
- **實際 Δv**：考慮阻力損失、重力損失
- **多級火箭**：級間 Δv 疊加

- 數值積分為 RK4 或簡化自適應（完整實現需專門庫）

### 驗證需求
- **實際工程應用**：需與試驗數據對比、專業工具交叉驗證
- **不確定度**：需執行 Monte Carlo 分析得到誤差帶
- **敏感度**：需執行敏感度分析找出主導誤差來源
- **可追溯性**：需記錄所有設計決策與驗證結果
